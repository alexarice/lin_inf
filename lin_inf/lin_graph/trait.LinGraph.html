Expand description

Functions for working with graphs

Implementing this trait allows the type to act as an undirected graph, and equips it with functions for working with graphs and their relation to linear inferences. Note that graphs defined with this trait do not know their size and many of the functions require this size to be passed in. This is done to save memory space when working with large numbers of graphs.

fn build<T>(f: T, number_vars: usize) -> Self where
    T: Fn(Node, Node) -> bool, 

Build a graph of size number_vars from a function f describing the edge data.

For all x and y less than number_vars we should have:

!assert_eq(build(f,number_vars).get(x,y),f(x,y))

and if x or y are greater than or equal to number_vars then we should have:

!assert_eq(build(f,number-vars).get(x,y), false)
